##  Tecnologias & Ferramentas

O site foi construído seguindo o conceito de *Single Page Application* (SPA) com otimização de performance e animações fluidas.

* **Frontend:** HTML5 Semântico, **Tailwind CSS** (v3.4).
* **Interatividade:** JavaScript (ES6+) Vanilla.
* **Animações:** * **AOS (Animate On Scroll):** Para efeitos de entrada de elementos.
    * **VanillaTilt.js:** Para efeito 3D interativo nos cards de projetos.
    * **CSS Keyframes:** Para background animado e efeitos de digitação.
* **Design:** Glassmorphism, Dark Mode automático (persistente) e layout Responsivo.

## Projetos em Destaque

O portfólio exibe projetos focados em arquitetura de software, padrões de projeto e backend robusto:

### Backend & Arquitetura (PHP/Laravel)
* **Belgare Elegancy (SaaS):** Sistema de agendamento completo em **PHP 8 Puro** (Sem Frameworks). Implementação manual de Router, arquitetura MVC estrita e segurança com Argon2i.
* **VioliEvents:** Plataforma Fullstack de gestão de eventos utilizando **Laravel 11** e **Tailwind CSS**. Foco em relacionamentos Eloquent (N:N) e refatoração de UI.
* **Product Manager API:** API RESTful construída manualmente com **Container de Injeção de Dependência**, Repository Pattern e Singleton.
* **PHP OOP Login:** Sistema de autenticação seguro focado em Programação Orientada a Objetos e Clean Code.

### Outras Tecnologias
* **ClientTasks:** API para gestão de tarefas desenvolvida em **Java Spring Boot** com persistência em MySQL.
* **Visão Computacional:** Algoritmo em **C++** utilizando a biblioteca **OpenCV** para análise e comparação de imagens.

##  Funcionalidades do Site

* **Dark Mode / Light Mode:** Alternância de tema com persistência local (localStorage).
* **Galeria de Mídia (Modal):** Visualização de screenshots e vídeos dos projetos sem sair da página.
* **Cards Interativos:** Efeito de inclinação 3D (Tilt) ao passar o mouse.
* **Fundo Dinâmico:** Animação de "Blobs" otimizada via GPU para não impactar a performance.
